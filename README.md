# digit-sum-power-cycles
Python code for computing cycles of f_k(n) = digit sum of n^k
# Digit Sum Power Cycles

This repository contains the Python implementation used in the paper:

"Ciclos da Soma dos Algarismos de Potências"

## Function

f_k(n) = sum of digits of n^k

## Features

- Computes fixed points
- Detects cycles
- Works with arbitrary precision integers

## Usage

Edit the last line in the script:

exibir_resultados(k)

Example:
exibir_resultados(7)
